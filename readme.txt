== Installation ==

1. Unzip the ZIP plugin file
2. Copy the dvdpedia2wp.php to your Wordpress Plugin Folger '/wp-content/plugins'
3. Copy the wordpress.txt to ~/Libery/Application Support/DVDPedia/Templates
4. Export als text in DVDPedia with the wordpress.txt template
5. Upload your export in the plugin menu
6. Create a new page with [dvdpedia] as page text
